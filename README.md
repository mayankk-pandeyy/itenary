# Itenary Coming Soon (React + Vite)

Minimal and modern React website for `itenary.in`.

## Development

```bash
npm install
npm run dev
```

Open the local URL shown in terminal.

## Production build

```bash
npm run build
npm run preview
```

## Deploy

### Vercel

1. Push this folder to GitHub.
2. Import repo in [Vercel](https://vercel.com/).
3. Build command: `npm run build`
4. Output directory: `dist`
5. Add domains: `itenary.in` and `www.itenary.in`
6. Copy DNS records from Vercel dashboard to your domain registrar.

### Netlify

1. Push this folder to GitHub.
2. New site from Git in Netlify.
3. Build command: `npm run build`
4. Publish directory: `dist`
5. Add custom domain and copy DNS records.
